# run-minver-action

Installs and runs MinVer using the CLI and conventions of repos in the Particular org:

* MinVer settings may exist in `src/Custom.Build.props`



## Usage

Basic:

```yaml
    steps:
      - name: Run MinVer
        uses: Particular/run-minver-action@v1.0.0
```
